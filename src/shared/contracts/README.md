# Shared Contracts

Tato slozka je kanonicka smlouva mezi frontendem a backendem.

Obsahuje:

- JSON schema pro hlavni datove objekty
- TypeScript typy pro frontend a sdilenou vrstvu
- source-scope model
- zaklad struktury odpovedi asistenta

Pravidlo:

- nove API shape se maji nejdriv propsat sem
- frontend i backend se maji ridit touto vrstvou
